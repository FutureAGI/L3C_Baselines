# L3C_Baselines/data
Generate the benchmark data for l3c baselines

## Usage
```shell
python gen_maze_record.py 
  --output_path OUTPUT_PATH
  --task_type TASK_TYPE
  --maze_type MAZE_TYPE
  --max_steps MAX_STEPS
  --n n1,n2,n3
  --density d1,d2,d3
  --n_landmarks l1,l2,l3
  --epochs EPOCHS
  --workers WORKERS
```
